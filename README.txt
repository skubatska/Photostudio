README
----------------------------
Programming language: Ñ++

This program allows storing information about services, customers, representatives, and ordering a photo studio. It also is using for adding new data to the file system, deleting and editing existing records, searching for existing registrations by date of birth.
The program provides for verification of the entered data for identity. Entering the same data, a user will get a warning and an opportunity to introduce new data. The program uses a comfortable and intuitive interface, organized menu, and submenu system.
All information is stored in the photo studio file system, in which, besides customer orders, other relevant background information is stored — for example, a list of photo studio services, customer data, and photo salon representatives.
